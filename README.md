🏍️ Touring Trip Pamulang - Karang HawuAplikasi web statis sederhana (mobile-friendly) untuk merencanakan dan memandu perjalanan touring sepeda motor dari Pamulang ke Pantai Karang Hawu, Sukabumi. Aplikasi ini dibuat agar mudah diakses melalui smartphone selama perjalanan.✨ Fitur Utama⏱️ Countdown Timer: Penghitung waktu mundur yang presisi menuju waktu keberangkatan (otomatis berubah teks saat waktunya tiba).🗺️ Jadwal Perjalanan (Itinerary): Timeline lengkap perjalanan dari berangkat, sesi istirahat, acara di pantai, hingga pulang.🎲 Ide Games: Rekomendasi permainan tanpa alat yang bisa dilakukan oleh pasangan di pantai.💰 Estimasi Biaya (Budget): Rincian perkiraan pengeluaran selama perjalanan (bensin, makan, tiket, dll).🎒 Smart Checklist: Daftar bawaan (packing list) interaktif. Status centang barang akan tersimpan secara otomatis di memori browser (localStorage), sehingga tidak akan hilang saat halaman di-refresh.🛠️ Teknologi yang DigunakanAplikasi ini sangat ringan karena tidak memerlukan instalasi backend atau build tools.HTML5: Struktur utama halaman.Tailwind CSS (via CDN): Styling utilitas untuk desain antarmuka yang modern dan responsif.FontAwesome (via CDN): Ikon-ikon menarik.Vanilla JavaScript: Logika untuk navigasi tab, countdown timer, dan penyimpanan checklist bawaan.🚀 Cara MenjalankanKarena ini adalah web statis, kamu hanya perlu menjalankan file HTML-nya secara langsung:Clone repositori ini ke komputer kamu:git clone [https://github.com/username-kamu/nama-repo-kamu.git](https://github.com/username-kamu/nama-repo-kamu.git)
Buka folder proyek.Buka (klik ganda) file index.html menggunakan browser favoritmu (Chrome, Safari, dll).Untuk pengalaman terbaik, buka halaman ini di smartphone atau gunakan mode Developer Tools (Mobile View) di desktop.⚙️ Kustomisasi (Cara Mengubah Konten)Jika kamu ingin menyesuaikan aplikasi ini untuk perjalanan lainnya:Mengubah Tanggal Keberangkatan:Buka index.html, cari kode Javascript di bagian bawah, dan ubah tanggal pada baris berikut:const countdownDate = new Date("June 21, 2026 05:00:00").getTime();
Mengubah Foto Banner:Aplikasi mencari file bernama foto.jpg di dalam folder yang sama. Cukup siapkan foto perjalananmu, beri nama foto.jpg, dan letakkan satu folder dengan index.html. Jika foto tidak ada, aplikasi akan menampilkan gambar placeholder bawaan.Mengubah Warna Tema:Aplikasi ini menggunakan warna tema kustom di konfigurasi Tailwind pada tag <head>. Kamu bisa mengubah warna hex di objek colors: { green_theme: { ... } } sesuai selera.Dibuat untuk rencana perjalanan santai dan penuh kenangan! Hati-hati di jalan dan selamat menikmati Sunset! 🌅
